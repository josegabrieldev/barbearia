<!-- 2. DESCRIÇÃO DO PROJETO -->
## 📋 Sobre o Projeto

O projeto é um site institucional criado para barbearias que desejam se destacar das demais tendo mais conversões de clientes, sendo mais profissionais e tendo uma vitrine apresentando seu negócio.

<!-- 3. PROBLEMA QUE O SITE RESOLVE -->
## 🔧 Problema que o Site Resolve

O barbeiro que trabalha solo enfrenta um desafio diário: interromper atendimentos para responder mensagens no WhatsApp, confirmar horários e agendar clientes.

O site centraliza essas informações publicamente — horários de funcionamento, serviços e canal de contato direto — reduzindo interrupções durante o trabalho e mensagens fora do horário.
